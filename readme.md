# Vercel SDK

This is a W.I.P.

Follow these steps to get started:

1. Clone the repo and install deps

```
$ git clone <this repo>
$ cd <cloned folder>
$ yarn
```

2. Build the project (optional)

```
$ yarn build
```

3. Run the tests

```
$ yarn test
```
